## Tonalität & Stil

- **Stil:** News-Native, seriös, klar strukturiert
- **Zielgruppe:** Leser der Augsburger Allgemeinen
- **Branche:** Mode / Lifestyle / Service
- **Keywords:** vertrauenswürdig, informativ, übersichtlich, hochwertig

## Einbettungskontext

- Wird in Augsburger-Allgemeine.de eingebettet (Header + Footer vom CMS)
- Kein eigenes `<html>`, `<head>`, `<body>` — nur Content-Block
- Keine eigene Navigation
- Keine Scroll-Animationen / JavaScript
- Alle Styles im `<style>`-Tag innerhalb des One-Pagers
